# ROYAL BOT

Bem-vindo ao **ROYAL BOT**, um bot customizado, desenvolvido para integrar e gerenciar funcionalidades em um servidor de jogos. Este bot automatiza operações importantes para a comunidade, facilitando a administração e a interação dos membros em tempo real.

## ✨ Visão Geral

O **ROYAL BOT** foi criado para atender às principais necessidades de servidores de jogos, incluindo:

- **Registro de Membros:** Automatiza o processo de registro e controle de entrada de novos membros no servidor.
- **Gestão de Jogos Transmitidos:** Identifica e define de forma dinâmica quais jogos estão sendo transmitidos, com base nas configurações realizadas pelos Administradores (ADMs).
- **Casas de Apostas Legais:** Fornece uma listagem atualizada das casas de apostas de futebol legalmente autorizadas no mercado.

## 🛠️ Funcionalidades

### Registro de Membros
O bot faz o registro automático dos membros que entram no servidor, podendo também coletar informações adicionais de acordo com a configuração.

### Definir Jogos Transmitidos
Administradores do servidor possuem comandos específicos para informar quais jogos estão sendo transmitidos no momento, facilitando o engajamento e a organização do público.

### Listagem de Casas de Apostas Legais
Permite listar, sob comando, as casas de apostas de futebol que atuam legalmente, oferecendo segurança e transparência para quem busca opções confiáveis no mercado.

## �� Como Usar

1. **Clonagem do Repositório**
   ```bash
   git clone https://github.com/YamSantosGX/ROYAL_BOT.git
   ```
2. **Configuração**
   - Adicione o bot ao seu servidor seguindo as instruções específicas do seu framework (Discord, Telegram, etc).
   - Configure as permissões e variáveis de ambiente necessárias conforme especificado nos arquivos de configuração do projeto.

3. **Execução**
   - Execute o bot com o comando correspondente ao seu ambiente (por exemplo, `node bot.js` para um bot em Node.js).

4. **Comandos Principais**
   - `!registrar`: Inicia o registro de um novo membro.
   - `!transmitindo <nome-do-jogo>`: Define qual jogo está sendo transmitido.
   - `!casas-apostas`: Exibe a lista atualizada das casas de apostas legais.
